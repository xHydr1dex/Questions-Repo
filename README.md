# Competitive Programming Solutions

Solutions to competitive programming problems, written in C++.

Each problem lives in its own folder containing the solution and an explanation.

## Structure

```
problem-name/
├── problem-name.cpp   # Solution
└── explanation.md     # Approach, complexity, notes
boilerplate.cpp        # Reusable template with common utilities
```

## Boilerplate

[boilerplate.cpp](boilerplate.cpp) contains reusable utilities:
- Modular arithmetic & fast exponentiation
- Combinatorics (nCr with precomputed factorials)
- Sieve of Eratosthenes
- DSU (Union-Find)
- GCD / LCM
